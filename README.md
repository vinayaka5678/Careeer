# Backend-Ready Structure

This folder is prepared for the DBMS mini project backend.

Suggested stack:
- Node.js + Express
- MySQL
- JWT authentication
- REST APIs for users, careers, quiz questions, quiz results, recommendations, saved careers, and admin analytics

Suggested modules:
- `controllers/`
- `routes/`
- `models/`
- `middleware/`
- `config/db.js`

The frontend currently uses local seed data so the project can run as a presentation-ready demo.
